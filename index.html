Primeira aula de HTML

O que é HTML?

Sigla para HyperText Markup Language - Linguagem de Marcação de Hipertexto —,o HTML é componente base da web.
Isso quer dizer que ele permite a construção de websites e a inserção de novos conteúdos, como imagens e vídeos, por meio dos hipertextos.

Como funciona o HTML?

  • O HTML é composto por uma série de elementos que indicam ao navegador como exibir o conteúdo.
  • Os elementos HTML rotulam partes do conteúdo, como "título", "parágrafo", "link", etc.
  • O HTML é interpretado por navegadores.
  • O HTML codifica caracteres de forma curta, utilizando o padrão UTF-8
  • O HTML utiliza banco de dados Web, permitindo o armazenamento de dados em web, local e cache.

Como criar um código HTML?

  1. Abrir um editor de texto sem formatação, como o Bloco de Notas
  2. Selecionar a opção "Novo" no menu "Arquivo"
  3. Digitar o código HTML
  4. Selecionar a opção "Salvar" no menu "Arquivo"
  5. Salvar o arquivo com a extensão ".html"

Os principais códigos em HTML

Os principais códigos em HTML são as tags, que são usadas para definir a estrutura e o contúdo de uma página.
Elas são divididas em:

Tags de formatação de texto funções
<b> Define um texto em negrito
<i> Define um texto em itálico
<ins> Sinaliza trechos incluídos
<del> Sinaliza trechos removidos
<mark> Sinaliza trechos removidos
<del> Destaca trechos de texto
<small> Define um texto menor
<supl> Define um sobrescrito
<sub> Define um subscrito


Tags de conteúdo funções
<p> Define um parágrafo
<br> Insere uma quebra de linha
<hr> Define um filete horizontal
<article> Representa um pedaço independente do conteúdo de um documento
<aside> Representa uma parte de conteúdo relacionada com a página
<command> Representa um botão, um radiobutton ou checkbox
<details> Descreve detalhes de um documento ou partes


Tags de estrutura e outras tags funções
<table> Delimita uma tabela
<tr> Indica as linhas
<td> Forma as colunas
<thead> Define o cabeçalho
<tbody> Define o corpo
<tfoot> Define o rodapé
<a> Cria um link para outra página
<img> Exibe uma imagem na página
<ul> e <ol> Criam listas não ordenadas e ordenadas, respectivamente
